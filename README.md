# Usuarios API

API REST desenvolvida em Spring Boot durante trilha de estudos.

## Tecnologias
- Java 17
- Spring Boot
- Spring Data JPA
- H2 Database
- Maven

## Funcionalidades
- CRUD de usuários
- Validações com Bean Validation
- Tratamento global de exceções
- Retorno de erros padronizado

## Como executar

mvn spring-boot:run
